# Ecommerce Full Stack Application

## Project Description
This is a full-stack ecommerce application built using Spring Boot and React (Vite).  
Users can browse products, add items to cart, and place orders.

## Tech Stack

Backend
- Java
- Spring Boot
- Spring Data JPA

Frontend
- React
- Vite
- JavaScript
- HTML
- CSS

Database
- h2 database

## Features
- User registration and login
- Product listing
- Add to cart
- Order management
- REST APIs

## Project Structure

ecommerce-project
 ├── backend (Spring Boot API)
 └── frontend (React + Vite)

## How to Run Backend

1. Navigate to backend folder
2. Configure database in `application.properties`
3. Run the Spring Boot application

## How to Run Frontend

1. Navigate to frontend folder
2. Run:

npm install  
npm run dev
